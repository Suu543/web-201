Dropdown 설명

  1. dropdown 버튼에는 배경색, 패딩으로 스타일을 준다.
  2. .dropdown class는 .dropdown-content class가 position: absolute일 때 부모 역학을 하기 위해 .dropdown class는 position: relative로 설정한다.
  3. .dropdown-content class는 실제 dropdown menu 요소를 가지고 있고, 기본값으로 display: none을 설정에 공간을 차지하지 않게 설정한다. 오직 hover event가 발생했을 때 display: block;으로 변한다.
  4. .dropdown-content 최소 길이는 min-width: 160px;로 설정했다. 자유롭게 변경해도된다.
  5. 작은 화면에서는 width: 100%;를 설정하고 overflow: auto를 해주면 스크롤이 가능하다.
  6. .dropdown-content의 z-index: 1;을 설정한 이유는 다른 요소의 위에 배치해 hover event 발생시 클릭에 문제가 없도록 만들기 위함이다.
